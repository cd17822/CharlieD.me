Python Syntax

All answers should be in one console separated by print statements regarding which question is which.


  1. Record and print the average of John’s grades (the actual number should only be stored once).
    -Math: 99
    -Science: 89
    -Psych: 96
    -Art: 92

  2. Do Q1 in 5 lines of code.

  3. Calculate how far away John’s average was from a 96 using the % operator.

  4. How far was it from a 65?

  5. Try with these grades now:
    -Geology: 100
    -Engineering: 99
    -Gym: 99
    -Social Studies: 99

    Hint: Check your answer! When you divide an integer (number without a decimal) by another integer, Python performs integer division, thereby cutting off anything past the decimal. In your equation if you turn a value into a float (number with a decimal), or insert a float into the equation Python performs standard division.
